# interactive-gpt-landing

A landing page for GPTengineer that lists down 3 key unique propositions about the product. 1 use case each on how developers & marketers can use it.

Make the landing page interactive and add cool animations and graphics to convey the point

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/interactive-gpt-landing.git
cd interactive-gpt-landing
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
